# Outlook Draft Manager

A Python GUI application to manage Outlook draft emails.

## Features
- Read data from Excel
- Save multiple configurations
- Preview and create draft emails
- Support for attachments and BCC
- Template variables

## Requirements
- Python 3.x
- Outlook installed (Windows)

## Installation
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the application:
```bash
python src/main.py
```
